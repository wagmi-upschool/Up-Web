# Feedback Form Rules

- Feedback soruları zorunlu değildir.
- Kullanıcı isterse bir soruyu boş bırakıp formu gönderebilir.
- Likert yıldız seçimi tekrar tıklanarak veya `Yıldızı kaldır` aksiyonuyla temizlenebilmelidir.
- Boş bırakılan cevaplar submit payload'ında `null` olarak gönderilmelidir.

Bu dosya, feedback akışında soru zorunluluğunun tekrar yanlışlıkla geri gelmesini önlemek için eklendi.
